# Customer Analytics & Segmentation for an E-Commerce Business

## Project Overview

This project focuses on analyzing customer behavior and sales performance for an e-commerce business using Data Analytics and Machine Learning techniques. The objective is to derive actionable business insights, improve customer retention, and identify high-value customer segments.

## Business Objectives

* Analyze sales and revenue trends
* Perform customer segmentation using RFM Analysis
* Identify customer groups using K-Means Clustering
* Conduct Cohort Analysis for retention measurement
* Evaluate product and category performance
* Assess the impact of discounts, GST, and marketing spend on revenue

## Dataset Information

The project utilizes multiple datasets:

* Online Sales Data (52,924+ transactions)
* Customer Demographics Data (1,468+ customers)
* Discount Coupon Data
* Marketing Spend Data
* Tax (GST) Data
